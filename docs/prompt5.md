Introduce a distinction between regular users and doctors. Doctors should have a separate registration panel — an expandable registration form button on the page. This is to support a booking system.

Doctors should be able to set their availability in a separate panel. The existing doctor address field should be replaced with a precise address. Users will then be able to select time slots for appointments. Each appointment should default to 20 minutes.

No payment gateway — only an appointment booking form with: date, time, patient details, and an optional description field. Appointments should also be reschedulable and cancellable.

Remove all existing links to Znany Lekarz — decouple the platform from that portal in favor of the in-house booking system.

Additionally, add a back-arrow navigation component so users can easily return to list views.
